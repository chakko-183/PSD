Bagging Classifer#

Bagging Classifier (singkatan dari Bootstrap Aggregating) adalah metode ensemble learning, yaitu teknik yang menggabungkan beberapa model (biasanya model yang sama) untuk mendapatkan hasil prediksi yang lebih stabil, akurat, dan tidak mudah overfitting.

Dalam klasifikasi, Bagging Classifier membuat banyak model (misalnya decision tree) pada subset data yang berbeda, lalu menggabungkan hasil prediksi dari semua model tersebut (melalui voting mayoritas).

Kata Bagging berasal dari dua konsep penting:#

  1. Bootstrap ‚Üí teknik pengambilan sampel acak dengan pengembalian dari dataset asli untuk membuat beberapa subset data pelatihan.

  2. Aggregating ‚Üí proses menggabungkan hasil prediksi dari setiap model agar menjadi satu keputusan akhir (misalnya voting mayoritas untuk klasifikasi).

Langkah-langkah Algoritma Bagging Classifier#

  1. Ambil beberapa subset data (bootstrap sampling) Dari dataset asli berukuran ùëÅ, dibuat ùëò subset data masing-masing juga berukuran ùëÅ, tetapi diambil secara acak dengan pengembalian (sehingga ada data yang mungkin muncul lebih dari satu kali, dan ada yang tidak muncul sama sekali).

  2. Latih model dasar (base estimator) Biasanya digunakan Decision Tree, tapi bisa juga model lain seperti SVM, KNN, atau lainnya. Setiap subset data dilatih dengan model yang sama tetapi di data yang berbeda.

  3. Lakukan prediksi dengan setiap model Setiap model menghasilkan prediksi terhadap data uji.

  4. Gabungkan hasil prediksi (aggregating)

  • Untuk klasifikasi, digunakan voting mayoritas ‚Üí kelas yang paling banyak dipilih oleh model-model individu akan menjadi hasil akhir.

  • Untuk regresi, digunakan rata-rata nilai prediksi.

import pandas as pd
import numpy as np
import matplotlib.pyplot as plt
import seaborn as sns
from sklearn.datasets import load_iris
from sklearn.preprocessing import LabelEncoder
from imblearn.over_sampling import SMOTE
from sklearn.model_selection import train_test_split
from sklearn.metrics import classification_report, accuracy_score, confusion_matrix
from sklearn.ensemble import BaggingClassifier
from sklearn.tree import DecisionTreeClassifier
from sklearn.decomposition import PCA
import warnings

warnings.filterwarnings('ignore')
sns.set_theme(style="whitegrid")

# --- SIMULASI DATA (Menyesuaikan dengan format kolom yang Anda gunakan) ---
# Memuat data Iris bawaan dan mensimulasikan ketidakseimbangan
iris = load_iris()
df = pd.DataFrame(iris.data, columns=iris.feature_names)
df['Species'] = iris.target_names[iris.target]

# Sengaja membuat data tidak seimbang (mengambil 15 sampel 'setosa')
df_setosa = df[df['Species'] == 'setosa'].sample(n=15, random_state=42)
df_other = df[df['Species'] != 'setosa']
df = pd.concat([df_setosa, df_other]).sample(frac=1, random_state=42).reset_index(drop=True)

# Menyesuaikan nama kolom agar sesuai dengan kode pengguna
df.columns = ["SepalLengthCm", "SepalWidthCm", "PetalLengthCm", "PetalWidthCm", "Species"]
# --- AKHIR SIMULASI ---

# Pisahkan fitur & target
X = df[["SepalLengthCm", "SepalWidthCm", "PetalLengthCm", "PetalWidthCm"]].values
y = LabelEncoder().fit_transform(df["Species"])
target_names = df["Species"].unique()

# Oversampling dengan SMOTE
smote = SMOTE(random_state=42)
X_res, y_res = smote.fit_resample(X, y)

print("Distribusi sebelum SMOTE :", dict(zip(*np.unique(y, return_counts=True))))
print("Distribusi sesudah SMOTE :", dict(zip(*np.unique(y_res, return_counts=True))))

# ======================================================
# 1. Bagging Classifier pada data sebelum SMOTE
# ======================================================
X_train, X_test, y_train, y_test = train_test_split(
    X, y, test_size=0.3, random_state=42, stratify=y
)

bag_before = BaggingClassifier(
    estimator=DecisionTreeClassifier(random_state=42),
    n_estimators=100,
    random_state=42,
    n_jobs=-1
)
bag_before.fit(X_train, y_train)
y_pred_before = bag_before.predict(X_test)
acc_before = accuracy_score(y_test, y_pred_before)

print("\n=== Hasil Bagging Sebelum SMOTE ===")
print(f"Akurasi: {acc_before:.4f}")
print(classification_report(y_test, y_pred_before, target_names=target_names))

# Confusion matrix
cm_before = confusion_matrix(y_test, y_pred_before)
plt.figure(figsize=(6, 5))
sns.heatmap(cm_before, annot=True, fmt="d", cmap="Reds",
            xticklabels=target_names, yticklabels=target_names)
plt.title(f"Confusion Matrix Sebelum SMOTE (Akurasi: {acc_before:.4f})")
plt.show()

# ======================================================
# 2. Bagging Classifier pada data sesudah SMOTE
# ======================================================
X_train_res, X_test_res, y_train_res, y_test_res = train_test_split(
    X_res, y_res, test_size=0.3, random_state=42, stratify=y_res
)

bag_after = BaggingClassifier(
    estimator=DecisionTreeClassifier(random_state=42),
    n_estimators=100,
    random_state=42,
    n_jobs=-1
)
bag_after.fit(X_train_res, y_train_res)
y_pred_after = bag_after.predict(X_test_res)
acc_after = accuracy_score(y_test_res, y_pred_after)

print("\n=== Hasil Bagging Sesudah SMOTE ===")
print(f"Akurasi: {acc_after:.4f}")
print(classification_report(y_test_res, y_pred_after, target_names=target_names))

# Confusion matrix
cm_after = confusion_matrix(y_test_res, y_pred_after)
plt.figure(figsize=(6, 5))
sns.heatmap(cm_after, annot=True, fmt="d", cmap="Greens",
            xticklabels=target_names, yticklabels=target_names)
plt.title(f"Confusion Matrix Sesudah SMOTE (Akurasi: {acc_after:.4f})")
plt.show()

# --- PCA VISUALIZATION ---
print("\n" + "="*50)
print("VISUALISASI PCA PERBANDINGAN DATA")

pca = PCA(n_components=2)
X_pca_original = pca.fit_transform(X)
X_pca_resampled = pca.transform(X_res)

Class_labels = target_names
colors = ["#FF5733", "#FFC300", "#6A0DAD"]

fig, axes = plt.subplots(1, 2, figsize=(15, 6))

# Plot data asli
for i, (color, label) in enumerate(zip(colors, Class_labels)):
    mask = y == i
    axes[0].scatter(X_pca_original[mask, 0], X_pca_original[mask, 1],
                    c=color, label=label, alpha=0.7, s=50)

axes[0].set_title('Data Asli (Sebelum SMOTE)', fontsize=14, fontweight='bold')
axes[0].set_xlabel(f'PC1 ({pca.explained_variance_ratio_[0]:.2%} variance)')
axes[0].set_ylabel(f'PC2 ({pca.explained_variance_ratio_[1]:.2%} variance)')
axes[0].legend()
axes[0].grid(True, alpha=0.3)

# Plot data setelah SMOTE
for i, (color, label) in enumerate(zip(colors, Class_labels)):
    mask = y_res == i

    # Membedakan sampel asli dan sintetis
    is_synthetic = np.isin(np.arange(len(X_res)), np.arange(len(X)), invert=True)

    # Plot sampel asli
    axes[1].scatter(X_pca_resampled[mask & ~is_synthetic, 0], X_pca_resampled[mask & ~is_synthetic, 1],
                    c=color, label=f'{label} (Asli)', alpha=0.9, s=50, marker='o')

    # Plot sampel sintetis
    axes[1].scatter(X_pca_resampled[mask & is_synthetic, 0], X_pca_resampled[mask & is_synthetic, 1],
                    c=color, label=f'{label} (Sintetis)', alpha=0.5, s=50, marker='x')

axes[1].set_title('Data Setelah SMOTE', fontsize=14, fontweight='bold')
axes[1].set_xlabel(f'PC1 ({pca.explained_variance_ratio_[0]:.2%} variance)')
axes[1].set_ylabel(f'PC2 ({pca.explained_variance_ratio_[1]:.2%} variance)')
axes[1].legend(loc='lower left')
axes[1].grid(True, alpha=0.3)

plt.tight_layout()
plt.show()
