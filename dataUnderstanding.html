Data understanding#

Data Understanding adalah proses mendalami data: struktur, sumber, kualitas, distribusi, dan keterbatasannya. Tujuannya memastikan data layak dipakai untuk analisis & modelling.

  1. Langkah-langkah Umum

  • Ingest & sampling ‚Äî ambil sample dataset (mis. 1k baris) untuk eksplorasi awal.

  • Profiling ‚Äî statistik dasar: count, unique, min/max, mean, std, percentiles.

  • Visualisasi awal ‚Äî hist, boxplot, scatter untuk deteksi pola & outlier.

  • Quality checks ‚Äî missing, duplicates, inconsistent formats, range checks.

  • Data dictionary / metadata ‚Äî jelaskan tiap kolom: tipe, allowed values, deskripsi.

  • Lineage & source validation ‚Äî darimana data? apakah ETL mengubahnya?

  • Privacy & compliance ‚Äî PII handling, GDPR/PDPA bila relevan.

  1. Dimensi Kualitas Data (detail)

  • Completeness (kelengkapan): ada missing/null?

  • Accuracy (ketepatan): apakah nilai mewakili kenyataan? (sulit verifikasi tanpa sumber eksternal)

  • Consistency (konsistensi): format kolom konsisten (tanggal, kode pos), business rules konsisten (ShipDate >= OrderDate)

  • Uniqueness: duplicate transactions / order IDs

  • Validity: apakah nilai di domain yang valid (mis. Segment ‚àà {Consumer, Corporate, Home Office})

  • Timeliness: data up-to-date sesuai kebutuhan analisis

  • Integrity / Referential: relasi antar tabel (jika ada) valid

  1. Data Dictionary (contoh ringkas untuk Superstore)

  • Order ID ‚Äî string, unik per order (mis. CA-2018-‚Ä¶)

  • Order Date ‚Äî date, format (YYYY-MM-DD)

  • Ship Date ‚Äî date

  • Ship Mode ‚Äî categorical (First Class, Second Class, Standard Class, Same Day)

  • Sales ‚Äî numeric (float), total sale amount

  • Quantity ‚Äî integer

  • Discount ‚Äî numeric (0.0 - 1.0) biasanya percent

  • Profit ‚Äî numeric (float)

try:
    conn = psycopg2.connect(
        user=USER,
        password=PASSWORD,
        host=HOST,
        port=PORT,
        dbname=DBNAME
    )
    print("‚úÖ Connection successful!")

cursor = conn.cursor()

create_table_query = """
CREATE TABLE IF NOT EXISTS superstore (
    Row_ID INT,
    Order_ID VARCHAR(50),
    Order_Date DATE,
    Ship_Date DATE,
    Ship_Mode VARCHAR(50),
    Customer_ID VARCHAR(50),
    Customer_Name VARCHAR(100),

    Country VARCHAR(50),

    State VARCHAR(50),

    Region VARCHAR(50),

    Category VARCHAR(50),

    Product_Name VARCHAR(255),
    Sales FLOAT,
    Quantity INT,
    Discount FLOAT,
    Profit FLOAT
);
"""

conn.commit()
print("‚úÖ Table ready!")

for _, row in df.iterrows():
